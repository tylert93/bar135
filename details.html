Overview

This website was made for a bar I used to work at and although they already have a website, it was a great project for me to attempt and gave me a lot of content to work with. Although the website has a lot of dynamic features, I wanted to create it without the help of Bootstrap to better my understanding of JavaScript and jQuery.
The site has two separate carousels for showcasing images of the bar and a div which displays a series of captions, all of which fade in and out of each other. It also has a section which shows the bars promotions for each day of the week and highlights the promotion for that particular day. All of these features were achieved using JavaScript.
The site also has a submission form which customers can use to send the bar an email. The form is fully functional and is linked to an email account on my phone, however I have not yet included any form validation. This was achieved using PHP. The site is also fully responsive to accommodate for a range of devices and screen sizes, collapsing the navigation bar and altering its layout.
This is the first project I ever created as a developer and I started making before I really knew anything about web development. I use it a vehicle to practice web development and I was very much learning on the job when I made it. For that reason, I can now look back and see how bloated and inefficient the CSS and JavaScript for it is.
